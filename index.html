Qui suis-je ?

Je suis étudiant en 4ème année de Sciences Informatiques à l'Université d'Ottawa (GPA 9.4/10), spécialisé dans le développement web Full Stack et l'ingénierie logicielle. Mon parcours combine une solide base académique avec plusieurs expériences pratiques en entreprise.

Actuellement basé à Ottawa/Gatineau, je développe des solutions robustes en utilisant l'écosystème React et Node.js.

Mes Compétences

Java Python React Node.js MongoDB SQL AWS (EC2, S3) Docker/CI-CD

Comment je travaille

  • Collaboratif & Agile : J'ai de l'expérience avec les méthodologies Agiles, les réunions fréquentes et l'intégration des feedbacks (testeurs/clients) pour améliorer l'UX.
  • Automatisation : Je cherche toujours à réduire le travail manuel et les erreurs humaines (ex: automatisation de requêtes SQL, pipelines CI/CD).
  • Full Lifecycle : Je m'implique de la conception (diagrammes relationnels) jusqu'au déploiement (AWS, Nginx) et à la maintenance.
